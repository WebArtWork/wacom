Usage

Bootstrap with provideWacom()

Use standalone providers by default. WacomModule remains available for older NgModule apps, but the intended entry point is provideWacom. 

import { provideWacom } from 'ngx-core';

export const appConfig = {
  providers: [provideWacom()],
};

Configuration

Centralize app-level defaults

Configure library-wide behavior once instead of scattering base URLs, storage prefixes, or route metadata policy across components. 

import { provideWacom } from 'ngx-core';

export const appConfig = {
  providers: [
    provideWacom({
      http: { url: 'https://api.example.com' },
      store: { prefix: 'waStore' },
      meta: {
        useTitleSuffix: false,
        applyFromRoutes: true,
        defaults: { links: {} },
      },
      network: {},
      socket: false,
      io: undefined,
    }),
  ],
};
